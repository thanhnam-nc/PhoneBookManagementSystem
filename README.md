# PhoneBook

A simple Flask-based phonebook web app with:

- User registration
- Login authentication
- Password reset flow
- SQLite database storage

## Run locally

### 1. Install dependencies

```bash
pip install -r requirements.txt
```

### 2. Start the app

```bash
python app.py
```

### 3. Install SQLite Viewer

- Go to **Extensions**
- Search for **SQLite Viewer**
- Click **Install**

### 4. Open the application

Open your browser and navigate to:

```text
http://127.0.0.1:5000
```

## Default admin account

A default admin account is created automatically on first run:
- Email: admin@phonebook.com
- Password: admin123
